프론트엔드란?

프론트엔드란 사용자의 화면에 보이는 웹 화면이다. 사용자의 컴퓨터에서 돌아가는 환경을 만드는 분야이다. 웹 사이트를 볼 때 브라우저에서 보는 모든 것을 나타내는 용어이고 여기에는 HTML, CSS 및 JavaScript가 포함된다. html은 태그를 제공하여 웹사이트 페이지의 구조를 생성한다. 웹 프로그래밍에서 뼈대를 담당한다. css는 글꼴 크기와 색상을 정의하여 사이트의 스타일을 정의한다. 웹 프로그래밍에서 피와 살, 디자인을 담당한다. javascript는 웹 사이트의 대화형 기능에 사용되는 클라이언트 측 스크립팅 언어이다. 웹 프로그래밍에서 근육을 담당한다. 프로그램이 움직일 수 있게 한다.

  • Vue

  • JS

이 있다.
